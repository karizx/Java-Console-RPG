# Java-Console-RPG
A simple, turn-based console RPG game in Java demonstrating Object-Oriented Programming (OOP) concepts.
Prosta, turowa gra RPG napisana w Javie. Projekt stworzony w celu praktycznego zastosowania zasad programowania obiektowego (OOP). Gra zawiera pętlę walki, interakcję z graczem (Scanner), losowe obrażenia i ciosy krytyczne. Zastosowano tu takie koncepcje jak Dziedziczenie i Polimorfizm (specjalna klasa Maga o innej mechanice ataku) oraz Hermetyzacja.
